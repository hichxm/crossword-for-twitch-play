# CrossWord for twitch chatDes mots fléches pour le chat d'une chaine twitch.![crossword](https://screenshotscdn.firefoxusercontent.com/images/a3862107-c69d-4fe8-a817-3599849f9f12.png)## Installation1) Télécharger le zip [ici](https://github.com/volca780/crossword-for-twitch-play/archive/master.zip)2) Décompressez le zip3) Créer un fichier `config.js`4) Mettez les informations suivant en modifiant le "username" et le mot de passe:	```js	config = {	    //Twitch Identifiant (use oauth token)	    username: "",	    password: "oauth:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",	    //Twitch chanel	    channel: "#volca780"	};	```5) Pour génerer le mot de passe connectez vous [ici](https://twitchapps.com/tmi/)6) N'oubliez pas de mettre votre chaine7) Mettez grâce au plugin Browser Source disponible sur OBS le lien ver le fichier `index.html`8) Bon jeu.## Créer mon mot fleché1) Installer `node.js` [ici](https://nodejs.org/en/download/)2) N'oubliez pas
